# imba_utils

```
yarn add illogikal/imba_utils
```

## waitForCondition

```
  import { waitForCondition } from 'imba_utils'
  ...
  await waitForCondition do someDelayedThingThatBecomesTrue

```

## sleep

```
  import { sleep } from 'imba_utils'
  ...
  await sleep 100 // waits for 100 milliseconds
  ...
```
